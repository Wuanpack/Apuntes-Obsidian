## ¿Qué es?

Es la confluencia de la inteligencia artificial y las tecnologías web que propone introducir descripciones explícitas sobre el significado de los recursos, para así permitir que las propias máquinas tengan un cierto nivel de comprensión sobre la web.

Es una evolución de la web tradicional que busca dar mayor significado al contenido online, permitiendo que los sistemas informáticos no solo muestren información, sino que la entiendan. El objetivo es que los motores de búsqueda, asistentes virtuales y otros agentes inteligentes puedan interpretar el contexto, las relaciones y el propósito del contenido, más allá de palabras clave sueltas.

La web semántica conecta datos estructurados entre sí, creando una red de significados basada en entidades, propiedades y relaciones. Esto se logra mediante tecnologías como:

* [[RDF (Resource Description Framework)]]
* [[OWL (Web Ontology Language)]]
* [[SPARQL (Lenguaje de consulta para grafos de datos)]]

Este enforque permite crear una web más inteligente, interoperable y centrada en el usuario, donde la información se adapta mejor a la intención real de búsqueda, y no solo a la coincidencia literal de términos.
